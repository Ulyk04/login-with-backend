## 🚀 How to Run the Project

### 1. Clone the Repository

```bash
git clone https://github.com/YOUR_GITHUB/YOUR_REPOSITORY.git
cd login-with-backend
cd client-side
npm install
npm start
cd ../server-side
npm install
node server.js
```
